# Java Console Utility App

## 📌 Description
This is a console-based Java application that provides multiple utilities in one program, including:
- Calculator (Addition, Subtraction, Multiplication, Division)
- Grade Evaluation System
- Multiplication Table Generator

The application is designed with a modular structure using multiple classes and includes input validation for better user experience.

---

## 🚀 Features
- Clean and structured code using OOP principles
- Input validation to prevent invalid entries
- Interactive console menu
- Multiple functionalities in a single application

---

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner (for user input)

---
